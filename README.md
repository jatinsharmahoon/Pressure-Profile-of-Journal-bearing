# Automotive Suspension System

## 📋 Project Overview
3D CAD model of an advanced suspension system with detailed assembly and technical documentation. This project showcases a complete independent double-wishbone suspension with dampers and anti-roll bars for high-performance vehicles.

## 🎯 Features
- Independent double-wishbone suspension design
- Integrated damping system for smooth ride quality
- Anti-roll bars for improved handling
- Optimized geometry for performance and comfort
- Fully parametric CAD model for easy modifications

## 📐 Specifications
- **Type**: Independent Double Wishbone
- **Travel**: 200mm (front), 180mm (rear)
- **Damper Type**: Telescopic hydraulic
- **Anti-roll Bar Diameter**: 22mm (front), 18mm (rear)
- **Material**: High-strength steel (4130)
- **Spring Rate**: 80 N/mm (tunable)

## 📁 Files Included
- `suspension-assembly.step` - Complete suspension assembly
- `suspension-drawing.pdf` - Engineering drawings
- `component-list.xlsx` - Bill of materials
- `geometry-analysis.pdf` - Suspension geometry analysis
- `images/` - 3D renders and cross-sections

## 🛠️ Tools Used
- Fusion 360
- CATIA

## 📝 Design Considerations
- Optimized for both on-road and light off-road conditions
- Modular design allows for easy upgrades
- Geometry tuned for minimal tire wear
